Создайте программу для управления оценками студента. Создайте три метода, которые модифицируют массивы разными способами:1., который:Создайте метод improveGradeПринимает массив целых чисел и индексУвеличивает оценку по этому индексу на 5 балловЕсли новая оценка превысит 100, установите её в 100Пример: improveGrade([70, 85, 90], 0) изменяет массив на [75, 85, 90]2.Создайте метод replaceFailingGrade, который:Принимает массив целых чиселЗаменяет любые оценки ниже 60 на 60Пример: replaceFailingGrade([55, 80, 45]) изменяет массив на [60, 80, 60]3., который:Создайте метод addBonusPointsПринимает массив целых чиселДобавляет 2 балла ко всем оценкамПример: addBonusPoints([88, 92, 95]) изменяет массив на [90, 94, 97]Выводите массив после каждого изменения в следующем формате:
After improving grade: [75, 85, 90]
After replacing failing: [60, 85, 60]
After adding bonus: [62, 87, 62]


Как работает каждый метод:

1. improveGrade(int[] grades, int index)

· Увеличивает оценку по указанному индексу на 5
· Проверяет: если больше 100 → ставим 100
· Пример: [70, 85, 55], index 0 → [75, 85, 55]

2. replaceFailingGrade(int[] grades)

· Проходит по всему массиву
· Находит оценки < 60 и заменяет на 60
· Пример: [75, 85, 55] → [75, 85, 60]

3. addBonusPoints(int[] grades)

· Добавляет 2 балла к каждой оценке
· Пример: [75, 85, 60] → [77, 87, 62]

Важно:

· Методы изменяют исходный массив (не создают новый)
· Используем цикл for для прохода по всем элементам
· Массивы передаются по ссылке, поэтому изменения видны в main.
